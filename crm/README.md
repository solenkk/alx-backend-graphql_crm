# Celery CRM Report Setup Guide

## Prerequisites

- Redis server installed and running
- Python dependencies installed

## Installation Steps

### 1. Install Redis

**Ubuntu/Debian:**

```bash
sudo apt update
sudo apt install redis-server
sudo systemctl enable redis-server
sudo systemctl start redis-server
```
